# Apple Liquid Glass

## Requirements
- MockFlow WireframePro MCP server (desktop on port 21194 or CLI install via npm)

## Author
MockFlow

## Category
design

## Output Type
wireframe

## Instructions
Design screens following a modern iOS aesthetic with a liquid glass finish. The color palette centers on a soft silver-white background (#F2F2F7), pure white surfaces (#FFFFFF), and a vibrant primary blue (#007AFF) for accents. Text uses SF Pro, with headings in bold black (#000000) and body text in deep gray (#3C3C43). UI components feature extreme rounded corners of 20px for cards and 12px for buttons. Buttons are solid glossy pills with a subtle top-to-bottom gradient. Cards and containers appear as thick glass slabs with a soft ambient shadow and a very thin light-gray border. Navigation bars at the top are clean white with large titles. Input fields are rounded rectangles with a subtle inset feel. The overall layout is spacious and breathable, mimicking the high-fidelity depth found in modern Apple software interfaces. Icons are rounded and colorful. The design should feel premium, fluid, and polished.

## Input examples
Describe a mobile screen (e.g. "a mobile smart home controller" or "a mobile fitness tracking dashboard")

## Output
Call the `render_wireframe` MCP tool from MockFlow WireframePro to render the result.
